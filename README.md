#  🖥️PyOS

Simulador de Sistema Operacional desenvolvido em Python com foco educacional, utilizando conceitos básicos de Sistemas Operacionais.

⸻

 # 📌Funcionalidades

Criação de processos,
Escalonamento Round Robin,
Bloqueio e desbloqueio de processos,
Detecção de deadlock,
Comunicação entre processos,
Fork de processos,
Gerenciamento de recursos,
Shell interativo,

⸻

#⚒️Tecnologias

Python
______ 

#🎯Objetivo

Projeto criado para praticar conceitos de Sistemas Operacionais de forma simples e prática.
